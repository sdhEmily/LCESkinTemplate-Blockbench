# LCESkinTemplate-Blockbench
Simple LCE Skin Template loader for Blockbench


# How to use?
In Blockbench go to **File** > **Plugins** > **Load Plugin from URL** and paste in this link.\
`https://raw.githubusercontent.com/sdhEmily/LCESkinTemplate-Blockbench/refs/heads/main/lce_skin_template.js`\
Then you can go to **File** > **New** > **Legacy Console Skin**
